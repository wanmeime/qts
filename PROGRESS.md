# 量化交易策略进度看板

## 核心策略 — 缠论底分型突破策略

### 当前状态：回测验证完成 ✅

### 策略规则

| 规则 | 描述 |
|------|------|
| **买入条件** | 底分型突破第三根K线高点（third_high），买入价不超过 third_high × 1.05 |
| **买入时机** | 不限时间，只要涨幅不超过5%即可买入 |
| **止盈策略** | 盈利30%后减仓一半，剩余仓位继续持有 |
| **止损策略** | 动态止损，基于底分型振幅自适应设置止损位 |
| **成交量过滤** | 当日成交量 > 5日均量的1.5倍 |

### 策略亮点

1. **涨幅限制过滤**：突破后涨幅超过5%放弃买入，避免追高
2. **30%止盈减仓**：锁定部分利润，释放资金重新投资
3. **资金复用效应**：减仓释放的资金通过重新投资产生198.23%的收益率
4. **动态止损**：基于底分型结构自适应设置止损位
