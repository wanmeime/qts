### 二、共享策略配置 strategy_config.py

回测和盯盘共用一套参数：买点优先级(二买>类二买>三买)、成交量倍数1.5、止盈30%减半、动态止损(振幅5%分界)。

### 三、回测系统

最终版 backtest_v2_takeprofit.py 定版，含验证器 verify_backtest.py。
绩效: +276.34% / 回撤17.32% / 夏普2.49 / 360个交易日。

### 四、盯盘模块 watchdog.py

新增 check_precise_buy_signals() 集成回测策略逻辑。
旧 缠论分析.py 已删除，统一到 chanlun_core。

### 十、财务数据接口（2026-07-02 新增）

| 模块 | 状态 | 说明 |
|------|:----:|------|
| `qmt_bridge.py /api/finance` | ✅ | 通过 xtquant 获取 PE/PB/市值，支持亏损趋势判断 |
| `static_analyzer.fetch_financial_info()` | ✅ | 优先 QMT，回退东方财富 |
| `static_analyzer.should_filter_stock()` | ✅ | 亏损股过滤 + 扭亏趋势放行 |
| `signal_monitor.py` 通知消息 | ✅ | 9处消息添加股票代码 |

**过滤逻辑**：
```
亏损（PE<0 或无EPS）
  ├─ loss_narrowing=True（亏损收窄中）→ 放行（扭亏潜力）
  └─ loss_narrowing=False/None → 过滤
盈利或数据不足 → 放行
持仓股 → 永远不过滤
```
