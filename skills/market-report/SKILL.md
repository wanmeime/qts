# 市场洞察报告生成 Skill

## 功能
从 Markdown 格式的市场分析报告生成带可视化图表的 HTML 报告。

## 使用方法

### 命令行
```bash
python _build/report_template.py <input.md> <output.html> [日期]
```

### Python 调用
```python
from _build.report_template import generate_html

# 读取 Markdown 报告
with open('report.md', 'r', encoding='utf-8') as f:
    md_text = f.read()

# 生成 HTML
html = generate_html(md_text, '2026年6月16日')

# 保存文件
with open('report.html', 'w', encoding='utf-8') as f:
    f.write(html)
```

## 报告结构

生成的 HTML 报告包含以下 7 个标准章节：

| 章节 | 内容 | 视觉元素 |
|------|------|----------|
| 一、指数数据 | 国内5大指数 + 涨跌幅 | 红涨绿跌表格 |
| 二、事件联动分析 | 情绪/资金/综合评判 + 事件详情 | 仪表盘 + 彩色评判框 |
| 三、核心分析 | 多方/空方观点 + 综合判断 | 红绿观点框 |
| 四、板块建议 | 关注 + 回避 | 红绿标注 |
| 五、操作策略 | 仓位建议 | 黄色策略框 |
| 六、风险提示 | 风险因素列表 | 红色警告框 |
| 七、看盘参考 | 上涨/下跌/横盘确认条件 | 三色信号框 |

## 视觉元素说明

### 仪表盘
- 🌡️ 情绪温度：红色=偏热，绿色=偏冷，橙色=中性
- 💰 资金流向：红色=偏流入，绿色=偏流出，橙色=中性

### 综合评判框
- 红色背景：市场偏多
- 绿色背景：市场偏空
- 深灰色背景：市场中性

### A股配色
- 红色 = 上涨/多方
- 绿色 = 下跌/空方

```markdown

### 3. 综合判断
...
```
