## 核心逻辑

1. **数据加载** → 读取A股全市场行情CSV
2. **股票过滤** → 排除ST、退市股、低成交额股票
3. **动量计算** → 计算近5日/10日/20日涨幅的综合动量得分
4. **排名选股** → 按动量得分排名，取前20只
5. **信号输出** → 输出CSV到30-信号目录

## 运行方式

```bash
# 生成今日信号

# 指定日期运行
/usr/bin/python3 /home/jiaod/qts/10-策略/动量轮动策略/信号生成.py --date 20260602
```
