# 动量轮动策略 - 策略说明

## 一、策略原理

动量轮动策略基于**动量效应（Momentum Effect）**——过去一段时间涨幅较大的股票，在未来短期内继续上涨的概率较高。

这是一种经典的趋势跟踪策略，核心思想是"追涨"：买入近期表现最强的股票，卖出表现转弱的股票，通过不断轮动持仓来捕获市场趋势。

## 二、策略流程

```
全市场股票（5524只）
    ↓
【第一步】基础过滤
    - 排除ST/*ST股票
    - 排除退市整理股票
    - 排除日均成交额 < 1000万的股票
    ↓
【第二步】动量因子计算
    - 计算近5日涨幅（短期动量）
    - 计算近10日涨幅（中期动量）
    - 计算近20日涨幅（长期动量）
    - 加权综合得分 = 5日×0.5 + 10日×0.3 + 20日×0.2
    ↓
【第三步】排名选股
    - 按综合动量得分降序排列
    - 取前20只股票
    ↓
【第四步】信号输出
    - 输出买入信号CSV
```

## 三、过滤规则详解

### 3.1 ST股过滤
- 排除名称中包含"ST"、"*ST"、"退市"的股票
- 这些股票通常有退市风险或涨跌幅限制不同

### 3.2 成交额过滤
- 排除日成交额低于1000万的股票
- 目的：确保流动性，避免买入后难以卖出
- 成交额单位：元（数据源中的成交额列）

### 3.3 价格过滤
- 排除最新价为0或NaN的股票（停牌/无交易）

## 四、动量因子计算

### 4.1 单期动量
```
动量_N日 = (最新价 - N日前收盘价) / N日前收盘价 × 100%
```

由于数据源为单日快照，使用涨跌幅近似：
- 5日动量 ≈ 近5个交易日的涨跌幅累计（需历史数据）
- 当仅有当日数据时，使用当日涨跌幅作为临时信号

### 4.2 综合动量得分
```
综合得分 = 0.5 × 动量_5日 + 0.3 × 动量_10日 + 0.2 × 动量_20日
```

权重设计说明：
- **短期权重最高（0.5）**：近期动量对短期预测更有效
- **中期权重次之（0.3）**：过滤短期噪音
- **长期权重最低（0.2）**：捕捉中期趋势

## 五、信号输出格式

| 字段 | 说明 |
|------|------|
| 排名 | 动量得分排名（1-20） |
| 代码 | 股票代码（如 sh600519） |
| 股票代码 | 纯数字代码 |
| 名称 | 股票名称 |
| 最新价 | 当前价格 |
| 涨跌幅 | 当日涨跌幅（%） |
| 成交额 | 当日成交额（元） |
| 换手率 | 当日换手率（%） |
| 动量得分 | 综合动量得分 |
| 信号日期 | 信号生成日期 |

## 六、风险提示

1. **动量反转风险**：动量效应可能在市场转折时失效
2. **追高风险**：策略本质是追涨，买入时可能已在高位
3. **成交额阈值**：1000万的阈值可能排除部分小盘股机会
4. **单日数据局限**：当前仅用单日快照，多日动量需要历史数据支持
