# 自选股监控模块

## 模块概述

| 项目 | 内容 |
|------|------|
| **模块名称** | 自选股监控 |
| **模块类型** | 信号监控 / 缠论分析 |
| **适用市场** | A股（沪深京） |
| **数据源** | AKShare + 本地K线缓存 |
| **状态** | 开发完成，待测试 |

## 功能描述

基于缠论对自选股列表进行实时监控和分析，识别买卖信号。

核心能力：
- **缠论分析**: K线包含处理、分型识别、笔划分、中枢识别
- **买卖点识别**: 二买、类二买、二卖等信号自动检测
- **MACD辅助**: 背驰检测、趋势确认
- **信号评分**: 综合多维度的信号强度评估
- **批量扫描**: 对自选股列表全部股票逐一分析

## 目录结构

```
自选股监控/
├── PROJECT.md              # 本文件 - 项目说明
├── config.yaml             # 参数配置文件
├── monitor.py              # 主入口 - 批量扫描与结果输出
├── 缠论分析.py              # 缠论核心分析模块
└── signal_generator.py     # 信号评分与输出模块
```

## 模块依赖

| 模块 | 用途 |
|------|------|
| `缠论分析.py` | 缠论分析核心逻辑（分型、笔、中枢、买卖点、MACD） |
| `signal_generator.py` | 信号评分、过滤、输出 |
| `monitor.py` | 主入口，串联数据获取、分析、输出 |

## 数据流

```
自选股列表 (watchlist.json)
    ↓ 市场过滤
K线数据获取 (本地缓存 → AKShare)
    ↓
缠论分析 (缠论分析.py)
    ├─ K线包含处理
    ├─ 分型识别 (顶/底)
    ├─ 笔划分
    ├─ 中枢识别
    ├─ MACD计算
    └─ 买卖点识别 (二买/类二买/二卖)
    ↓
信号评分 (signal_generator.py)
    ├─ 买点类型得分
    ├─ MACD确认
    ├─ 量能确认
    └─ 趋势得分
    ↓
输出结果 (CSV + 终端)
```

## 信号类型

| 信号 | 中文名 | 说明 |
|------|--------|------|
| `buy_2` | 二类买点 | 一买后的回调低点，右侧交易 |
| `buy_2_like` | 类二买 | 上涨趋势中第一个上涨中枢的横盘突破 |
| `sell_2` | 二类卖点 | 反弹未破前高 |

## 使用方法

```bash
# 扫描全部自选股
/usr/bin/python3 /home/jiaod/qts/10-策略/自选股监控/monitor.py

# 只分析单只股票
/usr/bin/python3 /home/jiaod/qts/10-策略/自选股监控/monitor.py --code 000518

# 设置最低信号分数
/usr/bin/python3 /home/jiaod/qts/10-策略/自选股监控/monitor.py --min-score 60

# 显示详细信息
/usr/bin/python3 /home/jiaod/qts/10-策略/自选股监控/monitor.py --verbose

# 指定日期
/usr/bin/python3 /home/jiaod/qts/10-策略/自选股监控/monitor.py --date 20260607
```

## 配置说明

详见 `config.yaml`，主要配置项：

| 配置 | 说明 | 默认值 |
|------|------|--------|
| `data.kline_days` | K线取样天数 | 120 |
| `data.adjust` | 复权方式 | qfq |
| `market.enabled_markets` | 启用的市场 | 上海A股、深圳A股 |
| `market.skip_markets` | 跳过的市场 | 美股、创业板 |
| `signal.min_score` | 最低信号分数 | 50 |
| `runtime.request_delay` | 请求间隔(秒) | 0.3 |
