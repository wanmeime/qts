# 多因子选股策略 - 策略说明

## 1. 策略目标

从A股全市场（约5500只）中，通过多因子打分模型筛选出综合评分最高的20只股票，生成每日选股信号。

## 2. 数据源

- **文件路径**：`/home/jiaod/qts/00-研究/数据源/缓存/A股全市场行情.csv`
- **数据字段**：代码、股票代码、名称、最新价、涨跌幅、成交额、换手率、市盈率、市净率、总市值、流通市值、交易所

## 3. 预处理 - 排除规则

以下股票不参与评分，直接剔除：

1. **ST股票**：名称中包含"ST"的股票
2. **退市股**：名称中包含"退"的股票
3. **负PE**：市盈率 < 0（亏损公司）
4. **负PB**：市净率 < 0（净资产为负）

## 4. 因子打分模型

### 4.1 打分原理

每个因子使用**分位数归一化**，将原始值映射到 0-100 分：

- **正向因子**（越大越好）：`分数 = (值 - 最小值) / (最大值 - 最小值) × 100`
- **负向因子**（越小越好）：`分数 = (最大值 - 值) / (最大值 - 最小值) × 100`
- **适中因子**（中间最优）：使用高斯衰减函数，距中间值越近得分越高

### 4.2 六大因子

| 因子 | 权重 | 打分方向 | 逻辑 |
|------|------|----------|------|
| PE（市盈率） | 20% | 负向（越低越好） | 低估值股票更有安全边际 |
| PB（市净率） | 15% | 负向（越低越好） | 低市净率意味着资产折价 |
| 换手率 | 15% | 适中（中间最优） | 过高说明投机过度，过低说明流动性差 |
| 总市值 | 20% | 正向（越大越好） | 大盘股更稳定，流动性更好 |
| 成交额 | 15% | 正向（越大越好） | 成交活跃的股票流动性好 |
| 涨跌幅 | 15% | 适中（接近0最优） | 避免追涨杀跌，选波动适中的 |

### 4.3 综合评分

```
综合得分 = PE分数 × 0.20 + PB分数 × 0.15 + 换手率分数 × 0.15
         + 市值分数 × 0.20 + 成交额分数 × 0.15 + 涨跌幅分数 × 0.15
```

## 5. 输出

- 按综合得分降序排列，输出**前20只股票**
- 输出文件：`/home/jiaod/qts/30-信号/多因子选股_YYYYMMDD.csv`
- 输出字段：代码、名称、最新价、涨跌幅、PE、PB、综合得分、排名

## 6. 风险提示

- 本策略基于历史截面数据打分，不构成投资建议
- 多因子模型存在因子失效风险
- 单日选股信号需结合趋势和风控体系使用
