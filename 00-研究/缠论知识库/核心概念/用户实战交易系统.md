## 二、选股流程

### 2.1 第一步：市场扫描

从当前市场中识别：
- **最热门的板块**
- **领涨关注度最高的股票**

### 2.2 第二步：缠论分析

结合缠论对候选股票进行分析：
- 买点判断（一、二、三类买点）
- **类二买判断**（用户创新）
- 卖点判断（一、二、三类卖点）
- 中枢位置分析
- 走势结构判断

### 2.3 第三步：筛选观察

**核心动作**：挑出**有可能在近期产生买点**的股票

→ 加入自选股，重点观察

### 2.4 选股池构建逻辑

```
热门板块 + 领涨股票
        ↓
缠论分析（买点/卖点/中枢）
        ↓
筛选：近期可能产生买点的股票
        ↓
过滤：排除亏损股
        ↓
估值分析（历史位置 + 同业对比）
        ↓
筛选：低估值优先
        ↓
选股池
```

## 三、交易原则

### 原则一：亏损股绝对不碰

> 亏损股票绝对不看，不选。

### 原则二：估值分析

重点查看股票的估值数据：
- 处于**历史高位**还是**低位**
- 在**同行业**中属于偏高还是偏低

### 原则三：低估值优先

优先选择**估值较低**的股票进入选股池

## 四、买入方法

### 4.1 多级别联立观察

```
日线级别：出现止跌/买点信号
    ↓
15分钟级别：观察小级别买点
    ↓
分时图：精准捕捉突破时机
```

### 4.2 倾向的买点：二买 / 类二买

#### 二买（第二类买点）

**定义**：第一类买点出现后，第一次次级别回调的低点。

**确认条件**：
1. 已经出现第一类买点
2. 从第一类买点开始的上涨结束后，出现第一次回调
3. 回调的低点不破第一类买点

**缠论定律支持**：大级别的第二类买点由次一级别相应走势的第一类买点构成。

#### 类二买（用户创新）

**定义**：上涨趋势中的第一个上涨中枢横盘突破后形成。

**形成条件**：
1. 中枢方向向上（上涨中枢）
2. 中枢后的走势向上突破中枢
3. 突破后形成新的向上走势

**与标准二买的区别**：
- 位置不同：类二买在中枢突破后，二买在回调低点
- 强度不同：类二买突破力度通常更强
- 安全性不同：类二买需要更多确认

**类二买的价值**：
- 类二买是两个上涨中枢之间连接线的部分
- 这段收益是**速度最快、力度最大**的一段
- 是上涨趋势中最佳的买入点

### 4.3 类二买的确认方法

```
上涨趋势中的第一个上涨中枢（上下上结构）
    ↓
出现小级别横盘震荡
    ↓
观察更小级别：
  • 底分型出现
  • 或突破小级别中枢上沿（通常是快速突破）
    ↓
突破小级别中枢上沿 → 类二买成立 → 买入
```

### 4.4 买入后的持有观察

```
买入后
    ↓
观察：能否突破大级别上涨中枢的上沿
    ↓
突破后，观察是否"站稳"
    ↓
站稳定义：回调没有再跌入中枢区间
    ↓
回调低点 = 第三类买点（三买）
```

### 4.5 三买的精确定义

> **回调不破前高 = 三买**

突破中枢上沿后，回调的低点没有跌回中枢区间。

## 五、卖出方法

### 5.1 卖出逻辑

**卖出是买入的反向操作**

### 5.2 二卖确认

```
上涨趋势
    ↓
出现一卖（最高点）
    ↓
回调
    ↓
再次上涨，但未突破一卖高点
    ↓
二卖形成
```

## 六、止损规则

### 6.1 双止损线规则

#### 第一止损线（常规止损）

**触发条件**：
- 二买底分型的**最低点被跌破**
- 跌破后**没有快速收回**
- → 触发止损

#### 第二止损线（强止损）

**触发条件**：
- 买入后股票下跌
- 跌破二买最低点后**继续向下**
- **突破了一买最低点**
- → **无论损失多大，必须止损**

### 6.2 止损逻辑图

```
买入（二买确认）
    ↓
下跌
    ↓
跌破二买底分型低点？
    ├─ 否 → 持有
    └─ 是 → 是否快速收回？
              ├─ 是 → 持有
              └─ 否 → 第一止损线触发 → 止损
                        ↓
              继续下跌，跌破一买低点？
              ├─ 否 → 等待观察
              └─ 是 → 第二止损线触发 → 强制止损
```

## 七、仓位管理

### 7.1 基本原则

- **完全基于技术点判断**
- 不过多考虑仓位控制
- 不过多考虑持有时机

### 7.2 操作方式

| 情况 | 操作 |
|------|------|
| **常规** | 全仓进，全仓出 |
| **特殊情况** | 涨幅较大且感觉有风险时，减仓一半 |

## 八、盘整操作（实战经验）

### 8.1 盘整可以操作

**传统观点**：盘整不参与

**实战经验**：在特定条件下，盘整是可以操作的

| 场景 | 操作方式 | 可行性 |
|------|---------|--------|
| 熊市中枢震荡 | 低点买、高点卖 | ✅ 实战验证可行 |
| 盘整中的短差 | 利用中枢上下沿 | ✅ 缠论第38课有说明 |

### 8.2 盘整的相对性

> "所有的股票，任何走势都是处于盘整中，只要把级别放得足够大"

- 日线的上涨，在周线可能是盘整
- 周线的上涨，在月线可能是盘整
- **盘整不参与是相对的，取决于操作级别**

### 8.3 类二买与盘整的关系

> "类二买本质上就是在盘整即将结束的时候才会产生的"

- 类二买出现在上涨中枢的横盘阶段
- 横盘即将结束时突破
- **这是对缠论的改良和创新**

## 九、技术架构

### 9.1 缠论 + MACD

- 缠论提供走势结构和买卖点框架
- MACD 提供趋势确认和背驰判断
- 两者结合使用是缠论完整体系的重要组成部分

### 9.2 多级别联立

```
日线级别：判断中期趋势
    ↓
15分钟级别：寻找精确买点
    ↓
分时图：捕捉突破时机
```

### 9.3 级别递归

缠论的级别体系在不同时间框架上同样适用：
- 月线 → 周线 → 日线 → 60分钟 → 30分钟 → 15分钟 → 5分钟 → 1分钟
