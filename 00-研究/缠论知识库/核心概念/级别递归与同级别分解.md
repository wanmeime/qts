# 级别递归与同级别分解

## 级别的定义（第17、20、53课）

### 显微镜比喻

级别是缠论中至关重要的概念，如同用不同倍数的显微镜观察同一个物体：

- 选定一个级别作为"显微镜"，就按该级别的定义来分析
- 什么级别的图和什么级别的中枢没有必然关系
- 日线图上可以找到1分钟中枢，1分钟图上也可以找到日线级别的背驰

### 级别体系

1分钟级别 -> 5分钟级别 -> 30分钟级别 -> 日线级别 -> 周线级别 -> 月线级别 -> 年线级别

### 级别递归

中枢的级别由构成它的次级别走势类型的级别决定：

- 由1分钟走势类型构成的中枢 = 1分钟中枢
- 由5分钟走势类型构成的中枢 = 5分钟中枢
- 由30分钟走势类型构成的中枢 = 30分钟中枢

### 缠中说禅走势级别延续定理一（第20课）

**在更大级别缠中说禅走势中枢产生前，该级别走势类型将延续。**

实际意义：要判断走势结束，首先要观察更大级别的中枢是否形成。

### 缠中说禅走势级别延续定理二（第20课）

**更大级别缠中说禅走势中枢产生，当且仅当围绕连续两个同级别缠中说禅走势中枢产生的波动区间产生重叠。**

恒星系统比喻：
- 中枢如同恒星，围绕它转动的行星构成恒星系统
- 两个同级别恒星系统要构成更大级别的系统，必然要其中外围行星之间发生关系

## 中枢级别扩张（第20课）

### 中枢扩张的条件

中枢的级别是逐步加大的，从一个5分钟的中枢，慢慢扩张成一个日线的，甚至变成周线的。

### 扩张方式

1. **中枢延伸**：围绕中枢的波动不断与中枢区间重叠，延伸到一定程度就升级
2. **中枢扩张**：新的同级别中枢的波动区间与前中枢的波动区间产生重叠

### 中枢延伸与趋势的区别

- **中枢延伸**：所有围绕中枢的前后两个次级波动都至少有一个触及中枢区间
- **趋势形成**：产生新的三次连续次级走势类型的重叠部分离开原来的中枢

### 中枢中心定理一（第20课）

走势中枢的延伸等价于任意区间[dn, gn]与[ZD, ZG]有重叠。若有Zn使得dn > ZG或gn < ZD，则必然产生高级别的走势中枢或趋势及延续。

### 中枢中心定理二（第20课）

前后同级别的两个缠中说禅走势中枢：
- 后GG < 前DD 等价于下跌及其延续
- 后DD > 前GG 等价于上涨及其延续
- 后ZG < 前ZD 且 后GG >= 前DD，或 后ZD > 前ZG 且 后DD <= 前GG，等价于形成高级别的走势中枢

## 同级别分解（第38课）

### 定义

把所有走势按一固定级别的走势类型进行分解。同级别分解具有**唯一性**。

### 核心思想

站在纯操作的角度，只要搞清楚如何判断背驰，选好适合的级别，当该级别出现底背驰时买入，顶背驰时卖出。

### 同级别分解规则

在某级别中：
- **不定义中枢延伸**
- **允许该级别上的盘整+盘整连接**
- 该级别以下的所有级别都**允许中枢延伸**，**不允许盘整+盘整连接**
- 该级别以上级别根本不考虑（所有走势都按该级别分解了）

### 与非同级别分解的区别

- 以前说不允许"盘整+盘整"是在非同级别分解方式下的
- 在同级别分解中，如果5分钟次级别延伸出6段，就当成两个30分钟盘整类型的连接

### 操作优势

这种分解对于机械化操作十分有利：
- 无所谓牛市熊市
- 如果分解的级别规定是30分钟，那么只要30分钟上涨就是牛市，否则就是熊市
- 市场被有效地肢解成一段段30分钟走势类型的连接

## 同级别分解操作程式（第38课）

### 向上段操作（先买后卖）

1. 从下跌背驰开始
2. 出现向上的第一段走势类型
3. 根据其内部结构判断背驰或盘整背驰结束点，先卖出
4. 必然有向下的第二段，两种情况：
   - 不跌破第一段低点 -> 重新买入
   - 跌破第一段低点 -> 如与第一段前的向下段形成盘整背驰，也重新买入，否则继续观望

### 向下段操作（先卖后买）

与向上段相反的操作流程。

## 同级别分解的多重赋格（第40课）

### 级别换档

当一个小级别的进入出现大级别的上涨时：
- 继续按小级别操作（代价累，精确度要求高）
- 升级为大级别操作基础上部分保持小级别操作（大资金更实用）

### 自动换档模式

在5分钟同级别分解中，只要从A0开始到某个At，使得A0+A1+...+At=B1+B2（30分钟级别的同级别分解），就可以继续按后一种分解操作。

### 多重赋格

对于大资金，可以变成N重层次的操作：
- 每一重对应着一定的资金与筹码
- 对应着不同的节奏与波动
- 如同赋格曲：简单的动机、旋律在N个层次上根据不同的转位、移位、对位等原则运动

## 走势类型连接的结合律（第33、36课）

### 结合律

走势分解、组合必须遵守结合律。

### 多义性（第33课）

多义性是与走势的当下性密切相关的，对已完成走势类型连接进行相应的分解。多义性不是含糊性，不是怎么分都可以。

### 多义性的来源

1. **中枢延伸的数量限制**：5分钟中枢延伸出现9段以上时，可解释为30分钟中枢
2. **模本的简略**：不同级别的图是对真实走势不同精度的模本
3. **多种合理释义**：a可看成围绕A中枢的波动，c可看成针对B的波动
