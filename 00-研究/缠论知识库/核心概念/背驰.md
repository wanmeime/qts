# 缠论背驰

## 定义（第15、24、37课）

在前后两个同向趋势中，当缠中说禅趋势力度比上一次的趋势力度要弱，就形成"背驰"。

### 核心原则

**没有趋势，没有背驰。** 在盘整中是无所谓"背驰"的。

### 缠中说禅趋势力度

前一"吻"的结束与后一"吻"开始由短线均线与长期均线相交所形成的面积。

---

## 背驰-买卖点定理（第24课）

**任一背驰都必然制造某级别的买卖点，任一级别的买卖点都必然源自某级别走势的背驰。**

### 含义

- 看到某级别的背驰，必然意味着要有逆转
- 逆转不一定永远，例如日线向上背驰制造卖点，回跌后在5分钟出现向下背驰制造买点，又可以重新上涨

---

## 背驰-转折定理（第29课）

某级别趋势的背驰将导致：
1. 该趋势最后一个中枢的级别扩展
2. 该级别更大级别的盘整
3. 该级别以上级别的反趋势

---

## a+A+b+B+c趋势背驰的精确判断条件（第37课）

### 结构要求

- A、B、C三段在一个大的趋势里
- B的中枢级别比A、C里的中枢级别都要大
- A段之前，一定是和B同级别或更大级别的一个中枢

### 判断标准

C段的走势类型完成时对应的MACD柱子面积比A段对应的面积要小，这时候就构成标准的背驰。

**向上趋势看红柱子，向下趋势看绿柱子。**

---

## 代码示例

```python
def check_beichi(segments_a, segments_c):
    """
    背驰判断
    
    比较A段和C段的MACD柱子面积
    """
    # 计算A段MACD柱子面积（向下看绿柱子）
    area_a = sum(abs(s.get('macd', 0)) for s in segments_a if s.get('macd', 0) < 0)
    
    # 计算C段MACD柱子面积
    area_c = sum(abs(s.get('macd', 0)) for s in segments_c if s.get('macd', 0) < 0)
    
    # C段面积小于A段，形成背驰
    if area_c < area_a:
        return True, area_a, area_c
    
    return False, area_a, area_c
```
