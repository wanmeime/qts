# 缠论线段

## 定义（第62课）

线段至少由三笔组成。

### 线段的两种基本形态

1. 从向上一笔开始的线段
2. 从向下一笔开始的线段

---

## 线段破坏

线段必须被线段破坏才算结束。单纯的一笔不能破坏线段，这避免了一些偶然因素对走势的干扰。

---

## 线段与笔的关系

- 线段包含至少三笔
- 线段的端点是某级别三类缠中说禅买卖点中的某一类
- 线段之间由笔连接

---

## 线段的精确划分标准（第67课）

### 特征序列

以向上笔开始的线段，可以用笔的序列表示：S1X1S2X2S3X3...SnXn。

- **向上线段的特征序列**：序列X1X2...Xn（向下的笔构成的序列）
- **向下线段的特征序列**：序列S1S2...Sn（向上的笔构成的序列）
- 特征序列的元素方向与线段方向相反
- **缺口**：特征序列两相邻元素间没有重合区间

### 标准特征序列

把每一元素看成一根K线，如同一般K线图中找分型的方法，也存在包含关系，可以进行非包含处理。经过非包含处理的特征序列称为标准特征序列。

### 特征序列的分型

参照一般K线图关于顶分型与底分型的定义：
- 以向上笔开始的线段，只考察**顶分型**
- 以向下笔开始的线段，只考察**底分型**

### 线段结束的两种情况

#### 第一种情况：无缺口

特征序列的顶/底分型中，第一和第二元素间**不存在**特征序列的缺口：
- 该线段在该分型的高/低点处结束
- 该高/低点是该线段的终点

#### 第二种情况：有缺口

特征序列的顶/底分型中，第一和第二元素间**存在**特征序列的缺口：
- 如果从该分型最高点开始的向下一笔开始的序列的特征序列出现底分型，那么该线段在该顶分型的高点处结束
- 如果从该分型最低点开始的向上一笔开始的序列的特征序列出现顶分型，那么该线段在该底分型的低点处结束

**强调**：在第二种情况下，后一特征序列不一定封闭前一特征序列相应的缺口，而且第二个序列中的分型不分第一二种情况，只要有分型就可以。

### 缠中说禅线段分解定理

**线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。而只要构成有重叠部分的前三笔，那么必然会形成一线段，换言之，线段破坏的充要条件，就是被另一个线段破坏。**

## 线段划分标准的再分辨（第71课）

### 特征序列元素的包含关系

特征序列的元素包含关系，前提必须是同一特征序列里的元素。不同特征序列之间的元素讨论包含关系没有意义。

特征序列元素的方向与对应的线段方向相反，因此不同方向的线段之间不可能存在包含关系。

### 特征序列分型的几种情况

1. **前一段未被笔破坏时**：可以存在前一特征序列的分型，序列元素的包含关系可以成立
2. **前一段被笔破坏时**：如果最早破坏的一笔不是转折点开始的第一笔，特征序列的分型结构也能成立（转折点前的最后一个特征序列元素与转折点后第一个特征元素之间肯定有缺口）
3. **转折点下来的第一笔就是最早破坏的笔**：这一笔处于待定状态，既不属于前一段的特征序列，也不属于后一段

### 从转折点开始的情况

从转折点开始，如果第一笔就破坏了前线段，进而该笔延伸出三笔来，其中第三笔破了第一笔的结束位置，那么新的线段一定形成，前线段一定结束。

### 第二种情况的简化处理

为什么第二种情况中第二特征序列不再分第一二种情况：

按极限思维，如果线段C对线段B还是第二种情况，那么线段C的区间在线段A特征序列缺口与线段B特征序列缺口之间。如此类推，总会出现一个线段使得对应前面的线段是回补特征序列缺口的。因此，在一串相对前一线段是第二种情况的线段串中，最终必然出现第一种情况的破坏。

---

## 缺口的处理（第64课）

- 顺着走势的缺口：包含在笔中，和一般走势没区别
- 逆着走势的缺口（如跳空）：必须当成一段
- 缺口可以看成是三个缺口的叠加，这样就有三笔以上

---

## 线段与笔的基本性质（第77课）

- 线段必须至少有三笔（两笔不能构成线段）
- 线段开始的那三笔必须有重合
- 线段中包含笔的数目都是单数的
- 线段必须被线段破坏才能确定完成
- 线段被线段破坏必须是被不同性质的线段破坏（向上线段不能被向上线段破坏）

---

## 分型-笔-线段的层级关系

```
K线（经过包含处理）
    |
    v
分型（顶分型、底分型）
    |
    v
笔（顶底之间至少一根独立K线）
    |
    v
线段（至少三笔）
    |
    v
中枢（至少三段次级别走势类型重叠）
    |
    v
走势类型（盘整、趋势）
```

## 代码示例

```python
def find_segments(bis):
    """
    线段识别
    
    规则：
    1. 线段至少由三笔组成
    2. 线段必须被线段破坏才算结束
    """
    if len(bis) < 3:
        return []
    
    segments = []
    i = 0
    
    while i < len(bis) - 2:
        # 检查是否有至少三笔
        if i + 2 < len(bis):
            # 判断方向
            if bis[i]['direction'] == 'up':
                # 向上线段：下-上-下 或 上-下-上
                if (bis[i]['direction'] == 'up' and 
                    bis[i+1]['direction'] == 'down' and 
                    bis[i+2]['direction'] == 'up'):
                    segments.append({
                        'start': bis[i]['start'],
                        'end': bis[i+2]['end'],
                        'direction': 'up',
                        'bis_count': 3
                    })
                    i += 2
                    continue
            
            elif bis[i]['direction'] == 'down':
                # 向下线段：上-下-上 或 下-上-下
                if (bis[i]['direction'] == 'down' and 
                    bis[i+1]['direction'] == 'up' and 
                    bis[i+2]['direction'] == 'down'):
                    segments.append({
                        'start': bis[i]['start'],
                        'end': bis[i+2]['end'],
                        'direction': 'down',
                        'bis_count': 3
                    })
                    i += 2
                    continue
        
        i += 1
    
    return segments
```

## 线段结束与买卖点定位

### 线段端点是买卖点的几何来源

缠论原文（第62课）明确：**"线段的端点是某级别三类缠中说禅买卖点中的某一类。"**

这意味着线段划分不是独立的技术游戏——**找到线段结束点，就是在找买卖点的位置。**

### 特征序列分型的买卖点信号

| 线段方向 | 关注的分型 | 信号含义 | 对应买卖点类型 |
|----------|-----------|---------|--------------|
| **向上线段**（↑） | 特征序列的**顶分型** | 上涨线段随时可能结束 → **卖点预警** | 第一类卖点、第二类卖点 |
| **向下线段**（↓） | 特征序列的**底分型** | 下跌线段随时可能结束 → **买点预警** | 第一类买点、第二类买点 |

### 三类买卖点的线段操作

#### 1. 第一类买卖点 = 趋势线段结束点
- **条件**：上涨/下跌趋势线段的最后一段出现{背驰}（特征序列分型 + MACD面积减小）
- **定位**：特征序列分型的位置即为第一类买卖点的价格区间
- **精确定位**：用区间套法在次级别线段上寻找更精确的买卖点

#### 2. 第二类买卖点 = 回调/反弹线段结束点
- **条件**：第一类买卖点后的第一次反向线段结束（回调/反弹）
- **定位**：用次级别的特征序列分型来精确识别该回调/反弹线段的结束点
- **定理**：大级别的第二类买点就是次级别第一类买点

#### 3. 第三类买卖点 = 离开中枢后的回试线段结束点
- **条件**：离开中枢的线段结束后，回试线段结束且不回到中枢内
- **关键**：**回试线段的结束位置**决定了是否构成第三类买卖点
  - 回试线段低点 > ZG → 第三类买点
  - 回试线段高点 < ZD → 第三类卖点

### 实战应用四步法

```
Step 1: 识别当前线段的运行方向（↑/↓）
Step 2: 构建特征序列，检查是否出现分型
    ├─ 无分型 → 线段继续，等待
    └─ 有分型 → 进入买卖点观察区
Step 3: 判断分型对应的是第几类买卖点
    ├─ 趋势末端背驰 → 第一类
    ├─ 第一次回调/反弹 → 第二类
    └─ 中枢后回试不破 → 第三类
Step 4: 在分型区间内执行买卖操作
```
