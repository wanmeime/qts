# 缠论走势类型

## 走势的三种分类（第15课）

所有走势都可以分解成三种基本情况，这是市场分析唯一值得依靠的基础。

### 上涨

最近一个高点比前一高点高，且最近一个低点比前一低点高。

### 下跌

最近一个高点比前一高点低，且最近一个低点比前一低点低。

### 盘整

最近一个高点比前一高点高，且最近一个低点比前一低点低；或者最近一个高点比前一高点低，且最近一个低点比前一低点高。

**注意**：所有上涨、下跌、盘整都建立在一定的周期图表上。在30分钟上的盘整，在日线图上可能就是上涨或下跌。抛开级别前提而谈论趋势与盘整是毫无意义的。

## 走势终完美（第17课）

### 缠中说禅技术分析基本原理一

**任何级别的任何走势类型终要完成。**

这个原理的重要性在于把静态的"所有级别的走势都能分解成趋势与盘整"转化成动态的、可以实用的"走势类型终要完成"。

### 缠中说禅技术分析基本原理二

**任何级别任何完成的走势类型，必然包含一个以上的缠中说禅走势中枢。**

### 走势分解定理一

任何级别的任何走势，都可以分解成同级别"盘整"、"下跌"与"上涨"三种走势类型的连接。

### 走势分解定理二

任何级别的任何走势类型，都至少由三段以上次级别走势类型构成。

## 走势类型的组合（第16课）

涨、下跌、盘整三种基本走势，有六种组合可能：

| 类型 | 组合 | 买入价值 |
|------|------|----------|
| 陷阱式 | 上涨+下跌 | 无 |
| 陷阱式 | 下跌+上涨 | 有 |
| 反转式 | 上涨+盘整+下跌 | 无 |
| 反转式 | 下跌+盘整+上涨 | 有 |
| 中继式 | 上涨+盘整+上涨 | 有 |
| 中继式 | 下跌+盘整+下跌 | 无 |

最有买入价值的标准程序："下跌+盘整+下跌"走势后出现第一类买点时介入。

## 级别

级别是缠论中至关重要的概念，如同用不同倍数的显微镜观察同一个物体。

### 级别体系

1分钟级别 -> 5分钟级别 -> 30分钟级别 -> 日线级别 -> 周线级别 -> 月线级别 -> 年线级别

### 级别与操作

- 什么级别的图和什么级别的中枢没有必然关系
- 选定一个级别作为"显微镜"，就按该级别的定义来分析
- 不同级别的买卖点意义不同，需在同一级别上研究

### 缠中说禅走势级别延续定理一

在更大级别缠中说禅走势中枢产生前，该级别走势类型将延续。

## 代码示例

```python
def classify_trend(segments):
    """
    走势分类
    
    根据高点和低点的变化判断走势类型
    """
    if len(segments) < 2:
        return "unknown"
    
    highs = [s['high'] for s in segments]
    lows = [s['low'] for s in segments]
    
    # 判断上涨
    is_up = all(highs[i] > highs[i-1] and lows[i] > lows[i-1] 
                for i in range(1, len(highs)))
    
    # 判断下跌
    is_down = all(highs[i] < highs[i-1] and lows[i] < lows[i-1] 
                  for i in range(1, len(highs)))
    
    if is_up:
        return "上涨"
    elif is_down:
        return "下跌"
    else:
        return "盘整"
```
