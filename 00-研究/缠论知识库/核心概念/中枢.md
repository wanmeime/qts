# 缠论中枢

## 定义（第17、18、20课）

### 基本定义

某级别走势类型中，被至少三个连续次级别走势类型所重叠的部分，称为缠中说禅走势中枢。

### 数学表达

次级别的连续三个走势类型A、B、C，分别的高、低点是a1\a2, b1\b2, c1\c2。

则中枢的区间为：
```
中枢 = [max(a2, b2, c2), min(a1, b1, c1)]
```

### 简化公式

```
ZG = min(g1, g2)    # 中枢上沿
ZD = max(d1, d2)    # 中枢下沿
中枢 = [ZD, ZG]
```

其中：
- GG = max(gn)：所有Z段高点的最大值
- DD = min(dn)：所有Z段低点的最小值

---

## 中枢的识别方法

### 基本识别

1. 找到至少三段连续的次级别走势类型
2. 计算三段的重叠区间
3. 如果重叠区间存在（高点大于低点），则构成中枢

### 中枢延伸

中枢形成后，后续的次级别走势如果与中枢区间有重叠，则属于中枢延伸。

**缠中说禅走势中枢中心定理一**：
走势中枢的延伸等价于任意区间[dn, gn]与[ZD, ZG]有重叠。

---

## 中枢的级别

### 级别递归

中枢的级别由构成它的次级别走势类型的级别决定：
- 由1分钟走势类型构成的中枢 = 1分钟中枢
- 由5分钟走势类型构成的中枢 = 5分钟中枢
- 由30分钟走势类型构成的中枢 = 30分钟中枢

### 级别与中枢的关系

- 什么级别的图和什么级别的中枢没有必然关系
- 日线图上可以找到1分钟中枢
- 1分钟图上也可以找到日线级别的背驰

---

## 中枢的扩展和升级

### 中枢升级的条件（第20课）

**缠中说禅走势级别延续定理二**：
更大级别缠中说禅走势中枢产生，当且仅当围绕连续两个同级别缠中说禅走势中枢产生的波动区间产生重叠。

### 中枢破坏定理（第18课）

某级别"缠中说禅走势中枢"的破坏，当且仅当一个次级别走势离开该中枢后，其后的次级别回抽走势不重新回到该中枢内。

---

## 中枢与走势类型的关系

### 盘整

某完成的走势类型只包含一个缠中说禅走势中枢 = 盘整

### 趋势

某完成的走势类型至少包含两个以上依次同向的缠中说禅走势中枢 = 趋势
- 方向向上 = 上涨
- 方向向下 = 下跌

**重要**：趋势中的中枢之间必须绝对不存在重叠。

## 代码示例

```python
def find_zhongshu(segments):
    """
    中枢识别
    
    规则：
    1. 至少三段连续的次级别走势类型
    2. 计算重叠区间
    """
    if len(segments) < 3:
        return []
    
    zhongshus = []
    i = 0
    
    while i < len(segments) - 2:
        # 取三段连续的走势
        s1, s2, s3 = segments[i], segments[i+1], segments[i+2]
        
        # 计算重叠区间
        zg = min(s1['high'], s2['high'], s3['high'])
        zd = max(s1['low'], s2['low'], s3['low'])
        
        if zg > zd:  # 存在重叠区间
            zhongshus.append({
                'zg': zg,  # 中枢上沿
                'zd': zd,  # 中枢下沿
                'gg': max(s1['high'], s2['high'], s3['high']),
                'dd': min(s1['low'], s2['low'], s3['low']),
                'segments': [s1, s2, s3]
            })
            i += 3  # 跳过已处理的三段
        else:
            i += 1
    
    return zhongshus
```
