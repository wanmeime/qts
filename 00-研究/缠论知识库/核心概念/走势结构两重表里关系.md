# 走势结构的两重表里关系

## 基本概念（第91课）

判断走势如同中医看病，未病而治的是第一等的，次之的是对治欲病，到已病阶段只能算是亡羊补牢。

### 级别与疾病的类比

| 级别 | 对应"疾病" |
|------|------------|
| 1分钟 | 小感冒 |
| 5分钟 | 小的感冒流行 |
| 30分钟/日线 | 次中级或中级调整（如肺结核） |
| 周线/月线 | 更严重的级别 |
| 季线/年线 | 植物人级别 |

### 未病-欲病-已病的界限

对应的界限就是相应级别的第一、二、三类买卖点。注意，对于上涨来说，踏空也是一种病，涨跌之病是相对的。

---

## 两种走势分解配件

### 第一种：能构成中枢的

包括线段、以及各种级别的走势类型。

### 第二种：不能构成中枢的

只有笔。笔是不能构成中枢的，这就是笔和线段以及线段以上的各种级别走势类型的最大区别。

---

## 表里关系的矩阵（第91课）

### 状态记录

可以随时给大盘开一个即时的病情记录，这个记录是一个矩阵：

- 按1、5、30、日、周、月、季、年的级别分类
- 矩阵有8行，每一行就是对应级别的状态数组
- 这矩阵可能的情况有4的8次方个，代表了走势所有可能的状态

### 状态的有限可变性

每一种状态后并不是随机到任何另一种状态的，可变的状态是极为有限的。从中可以分析出可能变化状态中出现最大可能赢利的转折状态，这种转折是必然的。

### 实际应用

一般人完全没必要去制造这样的机器，只需关心三个连续的级别（如1、5、30分钟），对应64种状态。

## 笔的表里关系（第91课）

### 笔的状态标记

用（1，1）表示向上的笔，（-1，1）表示向下的笔，（1，0）和（-1，0）表示笔的结束信号。

- （1，0）：向上笔结束
- （-1，0）：向下笔结束
- 这些信号是绝对明确、毫不含糊的，任何人都可以唯一地去确定

### 不同水平的操作方法（第93课）

以（1，0）为例：

#### 水平一：落袋为安型

- 震荡操作水平一般，胆子小，没时间
- 一个足够周期的（1，1）后出现（1，0），这意味着已经有足够的获利空间
- 最简单的做法：把成本先兑现出来，留下利润，让市场自己去选择
- 剩余筹码操作：如果出现（-1，1），向下笔结束后回来的向上笔只要不创新高，就把剩余筹码扔掉

#### 水平二：短差操作型

- 震荡操作水平比较好
- 利用（1，0）后必然出现的震荡进行短差操作
- 由于都是先卖后买，如果发现市场选择了（-1，1），最后一次就不回补了，完全退出

#### 水平三：精确操作型

- 技术最好的情况
- 利用区间套精确定位转折点
- 在盘口信息中发现符合区间套原则的背驰异动

## 更深层的表里关系

### 不同系统间的生克关系

光是笔这重表里关系，不足以精确地诊断市场走势。不同的表里关系之间还是有生克关系的，如同中医里不同系统间的生克关系一样。

### 多级别综合诊断

可能在笔这重关系中的"未病"，站在别的关系下就看出"已病"来了。因此必须综合多级别进行诊断。

---

## 操作意义

### 完全分类的威力

由于走势必完美，走势就是可以完全分类的。这种完全分类是级别性的，是有明确点位界限的，而不是粗糙的上下平概念。

### 当下性

完全分类使得任何的走势都成为可控的。这种可控并不需要任何人的预测或干预，而是当下直接地显现的，只需要根据当下的显示，根据自己的操作原则操作就可以。

## 完整8行状态矩阵模型（第91课）

### 矩阵结构

可以随时给大盘开一个即时的病情记录，这个记录是一个**矩阵**：

```
状态矩阵 = {
  1分钟: 状态,
  5分钟: 状态,
  30分钟: 状态,
  日线:  状态,
  周线:  状态,
  月线:  状态,
  季线:  状态,
  年线:  状态,
}
```

每一行就是对应级别的状态数组，各状态的可能取值有4种：(1,1), (1,0), (-1,0), (-1,1)。

### 状态空间的规模

每个级别有4种状态，8个级别：
- 总可能状态数 = **4⁸ = 65,536 种**
- 这代表了走势所有可能的宏观状态

### 状态的有限可变性

尽管状态总数很大，但**每一种状态后并不是随机到任何另一种状态的**，可变的状态是极为有限的。

从中可以分析出可能变化状态中出现最大可能赢利的转折状态，这种转折是必然的。

---

## 三个连续级别的简化使用（实战版）

一般人完全没必要去制造65,536种状态的机器。只需关心**三个连续的级别**（如1分钟、5分钟、30分钟），对应 **4³ = 64种状态**。

### 三级状态矩阵

```
当前状态 = { 1分钟: (X₁, Y₁), 5分钟: (X₂, Y₂), 30分钟: (X₃, Y₃) }
其中 X = -1 或 1，Y = 0 或 1
```

### 关键状态分析

#### 买入信号最强的状态

```
{1分钟: (1,0), 5分钟: (-1,1), 30分钟: (-1,1)}
```

**解读**：
- 30分钟向下笔运行中（大级别下跌）
- 5分钟向下笔运行中（中级别下跌）
- 1分钟出现向上笔结束（小级别底分型）

**含义**：各级别都在下跌末端，随时可能共振反弹。出现1分钟的(1,0)即底分型时，就是短线买入时机。

**操作**：这是"下跌+下跌+底分型"的买入结构，风险较小。

---

#### 卖出信号最强的状态

```
{1分钟: (-1,0), 5分钟: (1,1), 30分钟: (1,1)}
```

**解读**：
- 30分钟向上笔运行中（大级别上涨）
- 5分钟向上笔运行中（中级别上涨）
- 1分钟出现向下笔结束（小级别顶分型）

**含义**：各级别都在上涨末端，随时可能共振回调。出现1分钟的(-1,0)即顶分型时，就是短线卖出时机。

**操作**：这是"上涨+上涨+顶分型"的卖出结构。

---

#### 上涨中继状态

```
{1分钟: (-1,0), 5分钟: (1,1), 30分钟: (1,1)}
```

**解读**：
- 30分钟向上笔运行中
- 5分钟向上笔运行中
- 1分钟虽然顶分型，但只是小级别回调

**操作**：大方向向上，小级别回调是买入/加仓时机。等待1分钟(1,0)出现时补仓。

---

#### 下跌中继状态

```
{1分钟: (1,0), 5分钟: (-1,1), 30分钟: (-1,1)}
```

**解读**：
- 30分钟向下笔运行中
- 5分钟向下笔运行中
- 1分钟出现底分型，但只是小级别反弹

**操作**：大方向向下，小级别反弹是卖出/减仓时机。等待1分钟(-1,0)出现时减仓。

---

### 状态转换路径（邻接关系）

每种状态只能转换为有限的下一种状态。以(1,1)为例：

```
(1,1) → (1,0)  [向上笔延伸后出现顶分型 → 必然]
(1,0) → (1,1)  [顶分型失败，继续向上 → 可能]
(1,0) → (-1,1) [顶分型确认，开始向下笔 → 可能]
(-1,1) → (-1,0) [向下笔延伸后出现底分型 → 必然]
(-1,0) → (-1,1) [底分型失败，继续向下 → 可能]
(-1,0) → (1,1)  [底分型确认，开始向上笔 → 可能]
```

**必然路径**：任何状态最终都会转化为(1,0)或(-1,0)（分型必然出现）

**或然路径**：(1,0)接下来是继续向上还是转为向下，由次级别走势决定

## 表里关系的实战应用

### 每日盘面诊断

每天收盘后，可以做一个三级别矩阵诊断：

```
日期: 2026-06-18
状态矩阵:
  1分钟: (-1,0)  → 小级别顶分型，明日可能回调
  5分钟: (1,1)   → 中级别向上运行中
  30分钟: (1,1)  → 大级别向上运行中

诊断结果: 上涨中继状态，小级别回调是买入机会
操作计划: 等待1分钟(1,0)出现时逢低买入
```

### 三种操作水平（第93课）

以出现(1,0)（顶分型）为例：

| 水平 | 操作方式 | 适用人群 |
|------|---------|---------|
| **落袋为安型** | (1,1)后出现(1,0)，先兑现成本，留下利润筹码让市场选择 | 震荡操作水平一般、没时间 |
| **短差操作型** | 利用(1,0)后的震荡做短差，先卖后买，如转化为(-1,1)则最后一次不回补 | 震荡操作水平较好 |
| **精确操作型** | 利用区间套精确定位转折点，在盘口信息中发现符合区间套原则的背驰 | 技术最好 |

### 不同系统间的生克关系

光是笔这重表里关系，不足以精确地诊断市场。不同的表里关系之间还是有生克关系的，如同中医里不同系统间的生克关系一样。

**关键**：在笔这重关系中的"未病"，站在别的关系下可能看出"已病"来了。必须综合多级别进行诊断。
