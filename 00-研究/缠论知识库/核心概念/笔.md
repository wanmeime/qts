# 缠论笔

## 定义（第62、65课）

两个相邻的顶和底之间构成一笔。

### 笔的结构

- **上升的一笔**：底分型 + 上升K线（至少1根） + 顶分型
- **下降的一笔**：顶分型 + 下降K线（至少1根） + 底分型

---

## 笔的划分规则

### 规则一：顶底交替

笔必须是一顶一底交替：
- 先底后顶 = 向上一笔
- 先顶后底 = 向下一笔

### 规则二：至少一根独立K线

顶和底之间必须至少有一根独立的K线（不共用K线）。

- 如果顶和底共用一根K线，不算一笔
- 如果顶和底之间没有其他K线，最好也不算一笔
- 一笔的最基本图形：顶和底之间至少有一根K线

### 规则三：结合律

遵守结合律，不允许违反结合律的划分。

---

## 同类型分型的处理

### 两个顶之间没有底

第一个顶或底后的转折级别太小，不足以构成值得考察的对象，可以忽略其存在。

### 两个底之间没有顶

同上，中间的分型可以忽略。

---

## 笔的完全分解

所有的图形都可以唯一地分解为上下交替的笔的连接。

---

## 笔划分的唯一性证明（第77课）

笔的划分是唯一的。假设存在两个都满足条件的划分，且有所不同：

1. 两个划分从第N-1笔以前都是相同的，从第N笔开始出现第一个不同
2. 第N-1笔结束位置的分型性质对两个划分是一样的
3. 对于是顶的情况，第N笔的底对两个划分必然对应不同的底分型
4. 由于分型的划分是唯一的，这两个底在顺序上必然有前后高低之分

**如果前面的底高于后面的底**：前面的划分显然是错误的（一个底不经过一个顶后就有一个更低的底，这是典型的笔没完成的情况）

**如果前面的底不低于后面的底**：在下面一个顶分型出现前，如果有底分型低于前面的底，两种划分都不正确；如果没有，则必然有唯一的N+1笔划分。

---

## 笔划分的三步算法（第77课）

### 第一步：确定所有符合标准的分型

### 第二步：处理同类型分型

如果前后两分型是同一性质的：
- 对于**顶**：前面的低于后面的，只保留后面的，前面那个可以去掉
- 对于**底**：前面的高于后面的，只保留后面的，前面那个可以去掉
- 不满足上面情况的（如相等），都可以先保留

### 第三步：划分为笔

经过步骤二处理后，余下的分型如果相邻的是顶和底，就可以划为一笔。

**特殊情况**：如果相邻的性质一样，那么必然有前顶不低于后顶，前底不高于后底：
- 连续的顶后必然出现新的底，把连续顶中最先一个和新出现的底连在一起就是新的一笔
- 连续的底后必然出现新的顶，把连续底中最先一个和新出现的顶连在一起就是新的一笔
- 中间的那些分型都去掉

## 代码示例

```python
def find_bis(fractals, min_gap=4):
    """
    笔划分
    
    规则：
    1. 分型必须交替（顶-底-顶-底）
    2. 相邻分型之间至少有 min_gap 根处理后K线
    3. 顶分型的高点高于底分型的低点
    """
    if len(fractals) < 2:
        return []
    
    # 筛选有效分型，确保交替
    valid_fractals = [fractals[0]]
    
    for i in range(1, len(fractals)):
        curr = fractals[i]
        prev = valid_fractals[-1]
        
        # 同类型分型，取更极端的
        if curr['type'] == prev['type']:
            if curr['type'] == 'top':
                if curr['price'] > prev['price']:
                    valid_fractals[-1] = curr
            else:  # bottom
                if curr['price'] < prev['price']:
                    valid_fractals[-1] = curr
            continue
        
        # 检查最小间距
        gap = abs(curr['index'] - prev['index'])
        if gap < min_gap:
            continue
        
        valid_fractals.append(curr)
    
    # 构建笔
    bis = []
    for i in range(1, len(valid_fractals)):
        start = valid_fractals[i - 1]
        end = valid_fractals[i]
        
        if start['type'] == 'bottom' and end['type'] == 'top':
            direction = 'up'
        elif start['type'] == 'top' and end['type'] == 'bottom':
            direction = 'down'
        else:
            continue
        
        bis.append({
            'start': start,
            'end': end,
            'direction': direction
        })
    
    return bis
```

## 笔的补充说明（第65课）

### 笔的三类K线组合

经过包含关系处理后，没有包含关系的图形中，三相邻K线之间有完全分类：
1. 上升K线
2. 顶分型
3. 下降K线
4. 底分型

因此：
- 上升的一笔 = 底分型 + 上升K线（可以是无数根） + 顶分型
- 下降的一笔 = 顶分型 + 下降K线（可以是无数根） + 底分型

注意，上升、下降K线不一定都是3根，可以无数根，只要一直保持定义就可以。
