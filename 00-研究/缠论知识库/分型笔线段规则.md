# 分型、笔、线段规则

## K线包含处理（第62、65课）

### 定义
用[di, gi]表示第i根K线的最低和最高构成的区间。

当两根相邻K线存在包含关系时（一根K线的高低点全在另一根的范围内），需要进行包含处理。

### 处理规则

**向上时**（第n根与第n-1根比较，gn >= gn-1）：
- 顺次n个包含关系的K线组，等价于[max(di), max(gi)]的区间对应的K线
- 取两根K线的最高点为高点，两根K线低点中的较高者为低点
- 即"取并集"

**向下时**（第n根与第n-1根比较，dn <= dn-1）：
- 顺次n个包含关系的K线组，等价于[min(di), min(gi)]的区间对应的K线
- 取两根K线的最低点为低点，两根K线高点中的较低者为高点
- 即"取交集"

### 判断向上/向下
假设第n根K线满足与第n+1根的包含关系，而第n根与第n-1根不是包含关系：
- 如果 gn >= gn-1，则称第n-1、n、n+1根K线是**向上**的
- 如果 dn <= dn-1，则称第n-1、n、n+1根K线是**向下**的

### 顺序原则（结合律）
包含关系不符合传递律。必须按顺序处理：
1. 先处理第1、2根K线的包含关系，得到新K线
2. 用新K线与第3根比较
3. 如有包含关系，继续结合
4. 如无包含关系，按正常K线处理

### 完全分类
经过包含关系处理后，没有包含关系的图形中，三相邻K线之间有以下完全分类：
1. 上升K线
2. **顶分型**（第二K线高点最高，低点也最高）
3. 下降K线
4. **底分型**（第二K线低点最低，高点也最低）

## 分型（第62、65课）

### 顶分型
三根相邻K线中，中间K线的高点是三根中最高的，同时低点也是三根中最高的。

```
    [K2高]
   /      \
[K1高]   [K3高]
  |        |
[K1低]   [K3低]
   \      /
    [K2低]
```

### 底分型
三根相邻K线中，中间K线的低点是三根中最低的，同时高点也是三根中最低的。

```
[K1高]   [K3高]
  \      /
   \    /
    [K2高]
   /      \
[K1低]   [K3低]
   \      /
    [K2低]
```

### 分型的关键术语
- **顶分型的顶**：顶分型的最高点（简称"顶"）
- **底分型的底**：底分型的最低点（简称"底"）
- 顶分型的底和底分型的顶没有意义

### 分型的客观性
这种定义是唯一的，有统一答案的。即使缠中说禅本人弄错了，也就是错，没有任何含糊的地方。分析的结果与时间无关，与人无关，是客观的，不可更改的。

## 笔（第62、65课）

### 定义
两个相邻的顶和底之间构成一笔。

### 笔的结构
- **上升的一笔**：底分型 + 上升K线（至少1根） + 顶分型
- **下降的一笔**：顶分型 + 下降K线（至少1根） + 底分型

### 笔的划分规则

#### 规则一：顶底交替
笔必须是一顶一底交替：
- 先底后顶 = 向上一笔
- 先顶后底 = 向下一笔

#### 规则二：至少一根独立K线
顶和底之间必须至少有一根独立的K线（不共用K线）。

- 如果顶和底共用一根K线，不算一笔
- 如果顶和底之间没有其他K线，最好也不算一笔
- 一笔的最基本图形：顶和底之间至少有一根K线

#### 规则三：结合律
遵守结合律，不允许违反结合律的划分。

### 同类型分型的处理

#### 两个顶之间没有底
- 第一个顶或底后的转折级别太小，不足以构成值得考察的对象
- 第一个的顶或底可以忽略其存在

#### 两个底之间没有顶
- 同上，中间的分型可以忽略

### 笔的完全分解
所有的图形都可以唯一地分解为上下交替的笔的连接。

## 线段（第62课）

### 定义
线段至少由三笔组成。

### 线段的两种基本形态
1. 从向上一笔开始的线段
2. 从向下一笔开始的线段

### 线段破坏
线段必须被线段破坏才算结束。单纯的一笔不能破坏线段，这避免了一些偶然因素对走势的干扰。

### 线段与笔的关系
- 线段包含至少三笔
- 线段的端点是某级别三类缠中说禅买卖点中的某一类
- 线段之间由笔连接

### 缺口的处理（第64课）
- 顺着走势的缺口：包含在笔中，和一般走势没区别
- 逆着走势的缺口（如跳空）：必须当成一段
- 缺口可以看成是三个缺口的叠加，这样就有三笔以上

## 分型-笔-线段的层级关系

```
K线（经过包含处理）
    |
    v
分型（顶分型、底分型）
    |
    v
笔（顶底之间至少一根独立K线）
    |
    v
线段（至少三笔）
    |
    v
中枢（至少三段次级别走势类型重叠）
    |
    v
走势类型（盘整、趋势）
```

## 当下的客观性（第65课）

### 当下性
- 如果当下那些K线还没走出来，具体的分型就找不出来
- 一旦当下的K线走出来，就可以当下按客观标准唯一地找出相应的分型结构
- 当下的分析和事后的分析是一样的

### 级别选择
- 用什么图与以什么级别操作没有必然关系
- 用1分钟图也可以找出年线级别的背驰
- 看1分钟图并不意味着要玩超短线

### 精度选择
- 可以采取严格到小数后两位的精确度
- 也可以用两者相差0.5点内看成是一样的
- 唯一必须遵守的：精度一旦预设，就一定要一路保持

## 分型笔线段的完整算法流程

### 第一步：K线包含处理
1. 从第一根K线开始
2. 判断当前是向上还是向下
3. 检查下一根K线是否与当前K线有包含关系
4. 如有包含关系，按规则合并
5. 如无包含关系，输出处理后的K线

### 第二步：分型识别
1. 在处理后的K线上，检查每三根相邻K线
2. 判断是否构成顶分型或底分型
3. 记录所有分型及其位置

### 第三步：笔的划分
1. 将分型按时间顺序排列
2. 确保顶底交替
3. 检查相邻分型之间是否有足够的K线间隔
4. 如有同类型分型，取更极端的
5. 输出最终的笔

### 第四步：线段识别
1. 在笔的基础上，寻找至少三笔构成的线段
2. 判断线段是否被破坏（需要被线段破坏）
3. 输出线段及其方向

## 线段的精确划分标准（第67课）

### 特征序列

以向上笔开始的线段，可以用笔的序列表示：S1X1S2X2S3X3...SnXn。

- **特征序列**：序列X1X2...Xn称为以向上笔开始线段的特征序列；序列S1S2...Sn称为以向下笔开始线段的特征序列
- **缺口**：特征序列两相邻元素间没有重合区间，称为该序列的一个缺口
- **标准特征序列**：经过非包含处理的特征序列

### 特征序列的分型

参照一般K线图关于顶分型与底分型的定义，可以确定特征序列的顶和底：
- 以向上笔开始的线段的特征序列，只考察**顶分型**
- 以向下笔开始的线段，只考察**底分型**

### 线段结束的两种情况

#### 第一种情况

特征序列的顶/底分型中，第一和第二元素间**不存在**特征序列的缺口：
- 该线段在该分型的高/低点处结束
- 该高/低点是该线段的终点

#### 第二种情况

特征序列的顶/底分型中，第一和第二元素间**存在**特征序列的缺口：
- 如果从该分型最高点开始的向下一笔开始的序列的特征序列出现底分型，那么该线段在该顶分型的高点处结束
- 如果从该分型最低点开始的向上一笔开始的序列的特征序列出现顶分型，那么该线段在该底分型的低点处结束
- 后一特征序列不一定封闭前一特征序列相应的缺口
- 第二个序列中的分型，不分第一二种情况，只要有分型就可以

### 缠中说禅线段分解定理（第65课）

线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。而只要构成有重叠部分的前三笔，那么必然会形成一线段，换言之，线段破坏的充要条件，就是被另一个线段破坏。

## 线段划分的精度问题（第61课）

### 精度标准

- 可以采取严格到小数后两位的精确度
- 也可以用两者相差0.5点内看成是一样的
- 唯一必须遵守的：精度一旦预设，就一定要一路保持

### 新线段的标准

新线段的一个最重要标准就是一定要破坏老线段的结构：
- 下跌的线段，后高点一定低于前低点
- 上涨的线段，后低点一定高于前高点

---

## 线段划分的实用要点（第57、59课）

### 线段的长度问题

线段的长度没有固定标准：
- 有些线段延续上百点甚至更多，有些很短
- 关键看走出来的形态：如果任何低点比前一个高点都高，那么即使无限延续下去，也依然只是一线段
- 线段不存在一条直线走平的可能，因为线段至少呈现上下上或下上下

### 缺口与一字涨停

- 一字涨停无论如何延续，还是低于线段的级别，是最小级别的
- 缺口在最小分析级别下看成没有内部结构的线段

### 相邻线段方向

前后两线段间，不可能是同向的（同是向上或向下不可能构成两个前后相邻的线段）。

### 线段结束的判断方法（第59课）

线段的结束与大级别的走势段是一样的：
- **趋势中**：用背驰来确认线段结束
- **其他情况**：用盘整背驰来确认
- **突发性事件**：看第二类的买卖点

### 微弱结构的判断（第57课）

在判断线段时，需要注意区分微弱结构：
- 如果三段结构中第三段太微弱，把图形缩小后几乎看不到，则不构成线段
- 对比其他有明显第三段的结构，微弱结构只能看成是上下上的一个微小级别波动
- 在最小分析级别下，所有该级别以下的都可以看成没有内部结构的线段（包括跳空缺口和复杂的小波动）

### 线段与中枢的关系（第57课）

确定最小分析级别后：
- 该级别以下的都可以看成是线段
- 每一线段就是其次级别走势类型
- 三个线段重合部分就构成最小分析级别的中枢
