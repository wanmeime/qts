# 缠论中枢规则

## 中枢的定义（第17、18课）

### 基本定义
某级别走势类型中，被至少三个连续次级别走势类型所重叠的部分，称为缠中说禅走势中枢。

### 数学表达（第18课）
次级别的连续三个走势类型A、B、C，分别的高、低点是a1\a2, b1\b2, c1\c2。

则中枢的区间为：
```
中枢 = [max(a2, b2, c2), min(a1, b1, c1)]
```

### 简化公式
中枢的形成无非两种：
- 回升形成的：a1=b1, b2=c2
- 回调形成的：a2=b2, b1=c1

无论哪种情况，都可以简化为：
```
中枢 = [max(a2, c2), min(a1, c1)]
```

### Z走势段与中枢区间
与中枢形成方向一致的次级别走势类型称为Z走势段。

定义四个指标：
- **GG** = max(gn)：所有Z段高点的最大值
- **G** = min(gn)：所有Z段高点的最小值
- **D** = max(dn)：所有Z段低点的最大值
- **DD** = min(dn)：所有Z段低点的最小值

中枢区间：
```
ZG = min(g1, g2)    # 中枢上沿
ZD = max(d1, d2)    # 中枢下沿
中枢 = [ZD, ZG]
```

### 最低级别的中枢
对于最后不能分解的级别（如1分钟或5分钟），中枢定义为至少三个该级别单位K线重叠部分。

---

## 中枢的识别方法

### 基本识别
1. 找到至少三段连续的次级别走势类型
2. 计算三段的重叠区间
3. 如果重叠区间存在（高点大于低点），则构成中枢

### 中枢延伸
中枢形成后，后续的次级别走势如果与中枢区间有重叠，则属于中枢延伸。

**缠中说禅走势中枢中心定理一**：
走势中枢的延伸等价于任意区间[dn, gn]与[ZD, ZG]有重叠。若有Zn使得dn > ZG或gn < ZD，则必然产生高级别的走势中枢或趋势及延续。

### 中枢维持的充分必要条件
任何一个离开该中枢的走势类型都必须是次级别以下的并以次级别以下的走势类型返回。

---

## 中枢的级别

### 级别递归
中枢的级别由构成它的次级别走势类型的级别决定：
- 由1分钟走势类型构成的中枢 = 1分钟中枢
- 由5分钟走势类型构成的中枢 = 5分钟中枢
- 由30分钟走势类型构成的中枢 = 30分钟中枢
- 以此类推...

### 级别与中枢的关系
- 什么级别的图和什么级别的中枢没有必然关系
- 日线图上可以找到1分钟中枢
- 1分钟图上也可以找到日线级别的背驰

### 级别显微镜比喻
用不同倍数的显微镜看同一滴水，看到不同的精细度。级别之于走势也一样。

---

## 中枢的扩展和升级

### 中枢升级的条件（第20课）

**缠中说禅走势级别延续定理二**：
更大级别缠中说禅走势中枢产生，当且仅当围绕连续两个同级别缠中说禅走势中枢产生的波动区间产生重叠。

### 升级的具体方式
1. **中枢延伸**：围绕中枢的波动不断与中枢区间重叠，延伸到一定程度就升级
2. **中枢扩张**：新的同级别中枢的波动区间与前中枢的波动区间产生重叠

### 中枢延伸与趋势的区别
- **中枢延伸**：所有围绕中枢的前后两个次级波动都至少有一个触及中枢区间
- **趋势形成**：产生新的三次连续次级走势类型的重叠部分离开原来的中枢

### 中枢升级的判断（第24课答疑）
中枢的级别是逐步加大的。例如从一个5分钟的中枢，慢慢扩张成一个日线的，甚至变成周线的。

### 中枢破坏定理（第18课）
某级别"缠中说禅走势中枢"的破坏，当且仅当一个次级别走势离开该中枢后，其后的次级别回抽走势不重新回到该中枢内。

破坏的三种组合：
1. 趋势+盘整（最有力的破坏）
2. 趋势+反趋势
3. 盘整+反趋势

---

## 中枢与走势类型的关系

### 盘整
某完成的走势类型只包含一个缠中说禅走势中枢 = 盘整

### 趋势
某完成的走势类型至少包含两个以上依次同向的缠中说禅走势中枢 = 趋势
- 方向向上 = 上涨
- 方向向下 = 下跌

**重要**：趋势中的中枢之间必须绝对不存在重叠。

### 中枢定理一（第18课）
在趋势中，连接两个同级别"缠中说禅走势中枢"的必然是次级别以下级别的走势类型。

### 中枢定理二（第18课）
在盘整中，无论是离开还是返回"缠中说禅走势中枢"的走势类型必然是次级别以下的。

---

## 中枢与买卖点的关系

### 中枢下的买点
只有在下跌确立后的中枢下方才可能出现买点（第一类买点）。

### 中枢上的卖点
中枢延续中，中枢上不可能有买点，只可能有卖点。

### 第三类买卖点与中枢
- 第三类买点是中枢扩张或新生产生的
- 第三类卖点是中枢收缩或新生向下的结果

### 中枢震荡
在第二类和第三类买卖点之间，都是中枢震荡，不会有该级别的买卖点。参与其中的买卖，用的都是低级别的买卖点。

---

## 中枢的结合律与多义性

### 结合律
走势分解、组合必须遵守结合律。

### 多义性
多义性是与走势的当下性密切相关的，对已完成走势类型连接进行相应的分解。多义性不是含糊性，不是怎么分都可以。

### 同级别分解（第38课）
把所有走势按一固定级别的走势类型进行分解。同级别分解具有唯一性。

在同级别分解中：
- 不定义中枢延伸
- 允许该级别上的盘整+盘整连接
- 该级别以下的所有级别都允许中枢延伸，不允许盘整+盘整连接

---

## 中枢的恒星系统比喻（第20课）

缠中说禅走势中枢就如同恒星，和围绕该恒星转动的行星构成一个恒星系统。而两个同级别恒星系统要构成一个更大级别的系统，首先必然要至少是其中的外围行星之间发生关系。

某级别"缠中说禅走势中枢"的破坏，当且仅当一个次级别走势离开该中枢后，其后的次级别回抽走势不重新回到该中枢内。

破坏的三种组合：

3. 盘整+反趋势

---

## 中枢的"生住坏灭"（第18课）

中枢如同众生的生命周期：
- **生**：中枢的产生（三个连续次级别走势类型重叠）
- **住**：中枢的维持（离开和返回都是次级别以下）
- **坏**：中枢被破坏（次级别离开后回抽不回中枢）
- **灭**：中枢废弃

---

## 缠中说禅走势中枢中心定理二（第20课）

前后同级别的两个缠中说禅走势中枢：
- 后GG < 前DD 等价于下跌及其延续
- 后DD > 前GG 等价于上涨及其延续
- 后ZG < 前ZD 且 后GG >= 前DD，或 后ZD > 前ZG 且 后DD <= 前GG，等价于形成高级别的走势中枢

---

## 中枢的结合律与多义性（第33课）

### 多义性

多义性是与走势的当下性密切相关的，对已完成走势类型连接进行相应的分解。多义性不是含糊性，不是怎么分都可以。

### 纯中枢角度的背驰

站在B中枢的角度，b可以看成是向下离开中枢B，c可以看成是向上离开中枢B。所谓顶背驰，就是最后这个中枢，向上离开比向下离开要弱。

---

## 走势分解的重要原则（第54课）

### 两个同级别中枢之间必须有次级别连接

**核心规则**：两个同级别中枢之间必须有次级别的走势连接。

例如，g0d4 = g0d1+(d1g1+g1d2+d2g2)+（g2d3+d3g3+g3d4）这样的分解是不被允许的，因为括弧中的两个同级别中枢之间没有次级别的连接。

### 例外：三次级别构成中枢的扩展

如果三个连续的次级别走势类型构成中枢的扩展（形成更高一级别的中枢），则三个括弧相加之间没有次级别连接是允许的。这与两个同级别中枢的情况不同。

### 分解多样性与操作选择（第54课）

**分解原则**：
1. 所有的分解必须符合分解的原则（结合律、次级别连接等），不能胡乱分解
2. 对于具体操作，一定要选择当下有明确意义的分解
3. 例如是中枢震荡的，或有第三类买卖点的
4. 多种分解其实并不是什么麻烦事，反而是相互印证的好办法

### 分解的精度选择（第57、59课）

- 线段的确认取决于精度标准：精确到小数点后两位还是取整，对结果可能有影响
- 唯一必须遵守的：精度一旦预设，就一定要一路保持
- 新线段的最重要标准：一定要破坏老线段的结构（下跌线段的后高点一定低于前低点，上涨线段的后低点一定高于前高点）
