## 核心原理

缠论的理论基础建立在以下几个核心原理之上：

1. **走势终完美**（基本原理一）：任何级别的任何走势类型终要完成
2. **中枢存在性**（基本原理二）：任何级别任何完成的走势类型，必然包含一个以上的走势中枢
3. **走势分解定理**：任何级别的任何走势，都可以分解成同级别"盘整"、"下跌"与"上涨"三种走势类型的连接
4. **买卖点完备性**：市场必然产生赢利的买卖点，只有第一、二、三类
