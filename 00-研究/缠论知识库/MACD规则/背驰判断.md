# MACD背驰判断

## MACD判断背驰（第24课）

### 前提条件

用MACD判断背驰，首先要有两段同向的趋势。同向趋势之间一定有一个盘整或反向趋势连接，把这三段分别称为A、B、C段。

### 结构要求

- B的中枢级别比A、C里的中枢级别都要大
- A段之前，一定是和B同级别或更大级别的一个中枢
- A、B、C段在一个大的趋势里

### 背驰判断标准

B段的中枢一般会把MACD的黄白线回拉到0轴附近。C段的走势类型完成时对应的MACD柱子面积比A段对应的面积要小，这时候就构成标准的背驰。

**向上趋势看红柱子，向下趋势看绿柱子。**

### 实用技巧

看MACD柱子的面积不需要全出来，一般柱子伸长的力度变慢时，把已经出现的面积乘2，就可以当成是该段的面积。实际操作中根本不用回跌后才发现背驰，在上涨或下跌的最后阶段，判断就出来了。

## 缠中说禅MACD定律（第14课后回复）

**第一类买点都是在0轴之下背驰形成的，第二类买点都是第一次上0轴后回抽确认形成的。卖点的情况就反过来。**

## 标准两个中枢上涨的MACD形态（第25课）

一个标准的两个中枢的上涨，在MACD上会表现出以下形态：

### 第一段上涨

- MACD黄白线从0轴下面上穿上来
- 在0轴上方停留，形成第一个中枢
- 同时形成第二类买点

### 突破第一个中枢后

- MACD黄白线快速拉起
- 这往往是最有力度的一段
- 一切的走势延伸、MACD钝化都经常出现在这一段
- 一般在一个次级别的背驰中结束

### 形成第二个中枢

- MACD黄白线逐步回到0轴附近

### 突破第二个中枢后

- MACD黄白线以及柱子都再次重复前面的过程
- 但这次黄白线不能创新高
- 或者柱子的面积/伸长高度不能突破新高
- 出现背驰，结束两个中枢的上涨过程

## 背驰与盘整背驰（第24课）

### 背驰（趋势背驰）

- 发生在趋势中（至少两个同向中枢）
- 回跌一定至少重新回到B段的中枢里
- 技术上是绝对的，没有任何或然

### 盘整背驰

- 发生在盘整中（只有一个中枢）
- 如果C段不破中枢，一旦出现MACD柱子面积C段小于A段，其后必定有回跌
- 如果C段上破中枢，但MACD柱子面积小于A段，先出来，回跌有两种情况：
  1. 不重新跌回中枢 -> 构成第三类买点
  2. 重新跌回中枢 -> 继续该盘整

### 两者区别

| 特征 | 背驰（趋势背驰） | 盘整背驰 |
|------|-------------------|----------|
| 发生位置 | 趋势中 | 盘整中 |
| 回跌力度 | 至少回到B段中枢 | 不一定 |
| 操作意义 | 最重要 | 用于盘整中弄短差 |

## MACD与乖离判断（第25课）

### 超短线判断

MACD柱子伸长与乖离有关，大致就是走势和均线的偏离度。

### 乖离极限

- 在盘整中，柱子伸长到某个高度就一定回去
- 在趋势中，这个高度有一定极限
- 一旦触及乖离极限，特别是两次或三次上冲该极限，就会引发回调

## MACD分析流程总结

1. **确定分析级别**：选定操作级别（如30分钟）
2. **划分A/B/C段**：
   - A段：第一个中枢前的走势
   - B段：包含中间的较大中枢
   - C段：第二个中枢后的走势
3. **观察黄白线**：B段是否将黄白线回拉到0轴附近
4. **比较柱子面积**：C段的MACD柱子面积是否小于A段
5. **判断背驰**：面积减小 = 背驰形成
6. **结合级别**：多级别综合判断，确定背驰的级别和力度

## 代码示例

```python
def check_macd_beichi(dif, dea, macd, segments_a, segments_c):
    """
    MACD背驰判断
    
    比较A段和C段的MACD柱子面积
    """
    # 计算A段MACD柱子面积
    area_a = sum(abs(macd[i]) for i in segments_a if macd[i] < 0)
    
    # 计算C段MACD柱子面积
    area_c = sum(abs(macd[i]) for i in segments_c if macd[i] < 0)
    
    # 判断背驰
    if area_c < area_a:
        return True, area_a, area_c
    
    return False, area_a, area_c
```
