# 布林通道辅助判断规则

## 布林通道基础（第90课）

布林通道（BOLL，Bollinger Bands）由三条轨道线组成：
- **上轨** = 中轨 + 2 × 标准差
- **中轨** = N日简单移动平均线（通常N=20）
- **下轨** = 中轨 - 2 × 标准差

### 图形要素

- **上轨**：压力线，股价运行在上轨上方为超强状态
- **中轨**：多空分界线，股价在中轨上方为多头，下方为空头
- **下轨**：支撑线，股价运行在下轨下方为超弱状态
- **通道宽度**：反映波动率的大小，收窄=波动减小，张口=波动加大

## 超强区域与中阴状态（第90课）

### 超强区域判断

股价与布林通道的位置关系决定了走势的强弱状态：

| 位置 | 状态 | 含义 |
|------|------|------|
| 上轨以上运行 | **超强状态** | 一般中枢移动时会出现 |
| 上轨以下、中轨以上 | 多头正常区域 | 趋势运行中 |
| 中轨以下、下轨以上 | 空头正常区域 | 趋势运行中 |
| 下轨以下运行 | **超弱状态** | 急速下跌中 |

### 从超强区域转向中阴

- 从上轨上**跌回其下**：从超强区域转向一般性区域
- 从下轨下**涨回其上**：从超弱区域转向一般性区域
- **关键信号**：如果再次上涨或下跌创出新高/新低，但**不能重新有效回到超强/超弱区域**

→ 这意味着走势进入了**中阴状态**

---

## 中阴阶段结束时间判断（第90课）

**布林通道的收口，就是对中阴结束时间的最好提示。**

### 判断规则

1. **某一级别的布林通道收口** → 比这低级别的某个中阴过程要级别扩展或结束了
2. **收口完成 + 价格突破** → 一般都对应着有相应的第三类买卖点

### 中阴结束的布林通道形态

```
通道宽度（波动率）
    ↑
    |   ╱╲                      ╱╲
    |  ╱  ╲    ╱╲              ╱  ╲
    | ╱    ╲  ╱  ╲  中阴期    ╱    ╲
    |╱      ╲╱    ╲  ╱╲      ╱      ╲
    |               ╲╱  ╲    ╱
    |                    ╲  ╱
    |                     ╲╱
    +--------------------------------→ 时间
      趋势段    收口    变盘  新趋势
```

**关键观察点**：
1. 通道从扩张转为收窄 → 中阴开始
2. 通道持续收窄 → 中阴进行中（方向不明确）
3. 通道极度收窄后突然张口 → 中阴结束，方向选择完成
4. 张口的方向 = 中阴结束后的方向

## 第二类买卖点的辅助判断（第90课）

### 布林通道对第二类买卖点的滞后反应

进入中阴状态后，布林通道的**上轨和下轨都会滞后反应**。

### 判断方法

1. **第一次回跌或回升后再次向上或下跌时**，上轨和下轨才会转向
2. 这时转向的**上轨往往成为最大的阻力**，**下轨往往成为最大的支撑**
3. 第二类买卖点往往在转向后的上轨/下轨处被构造出来

### 实例：第二类买点的布林通道特征

```
价格
  ↑
  |        ╱────── 上轨（滞后转向，构成阻力）
  |    ╱──┤
  |  ╱    │第二类卖点？  ← 上轨压力
  |╱      │
  |       │
  |       ├────── 中轨
  |       │    ╱
  |       │  ╱
  |       │╱ ← 第二类买点 ← 下轨支撑
  |   ╱───┤
  |  ╱    └────── 下轨（滞后转向，构成支撑）
  |╱ 第一类买点
  +──────────────────────────────→ 时间
```

**注意**：布林通道是辅助工具，不能单独使用，必须结合走势结构和买卖点分析。

## 布林通道与中枢的关系

### 中枢边界与布林通道

- **中枢上沿（ZG）** 通常与布林通道上轨附近重合
- **中枢下沿（ZD）** 通常与布林通道下轨附近重合
- **中枢中轴** 通常与布林通道中轨一致

### 实战观察

当价格回到中轨附近时：
- 如果这是中枢震荡过程中的回拉，属于正常调整
- 中轨在上升趋势中构成支撑，在下降趋势中构成压力

### 布林通道的级别对应

不同级别的走势对应不同周期的布林通道：

| 操作级别 | 布林通道周期 | 中阴关注级别 |
|---------|-------------|-------------|
| 1分钟 | 5分钟布林通道 | 1分钟中阴 |
| 5分钟 | 30分钟布林通道 | 5分钟中阴 |
| 30分钟 | 日线布林通道 | 30分钟中阴 |
| 日线 | 周线布林通道 | 日线中阴 |

> **注意**：不是1分钟级别就一定要看1分钟的布林通道。例如一个5分钟的中阴过程，对应的是看30分钟的布林通道。

## 布林通道结合MACD的综合判断

### 布林通道 + MACD 双指标配合

| 信号组合 | 含义 | 可靠性 |
|---------|------|-------|
| 价格触上轨 + MACD顶背驰 | 顶部确认，卖点 | ★★★★★ |
| 价格触下轨 + MACD底背驰 | 底部确认，买点 | ★★★★★ |
| 布林收口 + MACD回拉0轴 | 中阴阶段，等待方向 | ★★★ |
| 布林张口 + MACD金叉突破0轴 | 上涨趋势启动 | ★★★★ |
| 布林张口 + MACD死叉跌破0轴 | 下跌趋势启动 | ★★★★ |

### 判断流程

```
判断当前所处阶段
    ↓
布林通道收口？───是──→ 中阴阶段，等待方向选择
    │                       ↓
   否                    观察第三类买卖点
    │                       ↓
    ↓                    布林张口 + 方向确认
价格在轨外运行？───是──→ 趋势运行中
    │                       ↓
    ↓                    结合MACD判断是否背驰
   价格在轨内（震荡）
    ↓
   结合中枢区间操作
```
