# 缠论类二买

## 定义

上涨趋势中的第一个上涨中枢横盘突破后形成。

### 识别条件

1. 中枢方向向上
2. 中枢后的走势向上突破中枢
3. 突破后形成新的向上走势

---

## 操作意义

- 类二买是第二类买点的变体
- 出现在中枢突破后，可视为追涨信号
- 强度通常不如标准第二类买点

---

## 与第二类买点的关系

类二买是第二类买点的变体，出现在中枢突破后。与标准第二类买点相比：

- 位置不同：类二买在中枢突破后，第二类买点在第一类买点后的第一次回调
- 强度不同：类二买强度通常不如标准第二类买点
- 安全性不同：类二买需要更多的确认

---

## 代码示例

```python
def find_class2_buy(zhongshus, segments):
    """
    识别类二买
    
    类二买：上涨趋势中的第一个上涨中枢横盘突破后形成
    """
    results = []
    
    for zs in zhongshus:
        zg = zs['zg']  # 中枢上沿
        
        # 寻找向上突破中枢的走势
        for seg in segments:
            if seg['start'] > zs['end']:  # 在中枢之后
                if seg['direction'] == 'up' and seg['high'] > zg:
                    # 向上突破中枢
                    results.append({
                        'type': 'class2_buy',
                        'index': seg['index'],
                        'price': seg['high']
                    })
                    break
    
    return results
```
