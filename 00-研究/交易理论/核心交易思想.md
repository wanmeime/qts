# 核心交易思想与选股方法

> 基于十几年实战经验总结，以缠论为核心技术框架

---

## 一、投资背景

- **入市时间**：2013年至今，约13年经验
- **核心理论**：缠论
- **交易风格**：技术分析 + 估值判断

---

## 二、缠论：核心理论基础

### 2.1 缠论的本质

缠论是一个**科学解读市场**的工具，而不是预测工具。

### 2.2 缠论的核心功能

1. **对已生成的走势形成完整体系**
2. **归类买点和卖点**
   - 一类买点、二类买点、三类买点
   - 一类卖点、二类卖点、三类卖点
3. **识别K线组合形态**
   - 通过特定K线组合判断缠论概念
4. **中枢理论**
   - 识别震荡区间和趋势方向

### 2.3 缠论的价值

- 不预测市场
- 对已发生走势进行科学归类
- 提供完整的买卖点判断框架

---

## 三、选股流程

### 3.1 第一步：市场扫描

从当前市场中识别：
- **最热门的板块**
- **领涨关注度最高的股票**

### 3.2 第二步：缠论分析

结合缠论对候选股票进行分析：
- 买点判断（一、二、三类买点）
- 卖点判断（一、二、三类卖点）
- 中枢位置分析
- 走势结构判断

### 3.3 第三步：筛选观察

**核心动作**：挑出**有可能在近期产生卖点**的股票

→ 加入自选股，重点观察

---

## 四、交易原则

### 原则一：亏损股绝对不碰

> 亏损股票绝对不看，不选。

### 原则二：估值分析

重点查看股票的估值数据：
- 处于**历史高位**还是**低位**
- 在**同行业**中属于偏高还是偏低

### 原则三：低估值优先

优先选择**估值较低**的股票进入选股池

---

## 五、选股池构建逻辑

```
热门板块 + 领涨股票
        ↓
缠论分析（买点/卖点/中枢）
        ↓
筛选：近期可能产生卖点的股票
        ↓
过滤：排除亏损股
        ↓
估值分析：历史位置 + 同业对比
        ↓
筛选：低估值优先
        ↓
选股池
```

## 六、核心理念总结

| 维度 | 内容 |
|------|------|
| **技术框架** | 缠论（走势归类、买卖点、中枢） |
| **选股起点** | 热门板块 + 领涨股票 |
| **核心筛选** | 近期可能产生卖点的股票 |
| **风险控制** | 亏损股绝对不碰 |
| **估值标准** | 历史低位 + 同业偏低 |
| **最终选择** | 低估值优先 |
